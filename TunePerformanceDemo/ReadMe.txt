Настройка проекта на локальном IIS
1. с помощью проекта TPD.DB развернуть БД с именем TunePerformanceDemo (http://weblogs.asp.net/gunnarpeipman/using-visual-studio-database-projects-in-real-life)
2. выполнить скрипт initialdata.sql для заполнения начальными данными
3. настроить проект в IIS (https://www.pluralsight.com/blog/tutorials/windows-server-2008-iis7)
4. добавить строку 127.0.0.1 tpd.com в host файл (c:\windows\system32\drivers\etc\hosts)
5. запустить проект http://tpd.com/
